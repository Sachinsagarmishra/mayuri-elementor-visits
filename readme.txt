Adds unique editable Elementor visit widgets for Mayuri sections.

== Description ==

Mayuri Elementor Visits adds a custom Elementor category named Mayuri Visits and a Services Grid Visit widget. The widget recreates the supplied services-grid design with editable cards, images, icons, text, links, colors, gradients, typography, spacing, and responsive settings.

== Installation ==

1. Upload the plugin zip from WordPress Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Make sure Elementor is installed and active.
4. Edit a page with Elementor and search for Services Grid Visit.
